Zombie Survival

Zombie Survival is an engaging game where players must survive waves of zombies by strategically using resources and defenses. The goal is to last a minute while managing limited supplies and upgrading defenses.

Key Features & Technical Work

  • Real-time game loop using Java Swing timers.
  • Dynamic zombie spawning with increasing difficulty.
  • Resource management system for upgrades and defenses.
  • Collision detection for player-zombie interactions.
  • Graphical user interface designed with Java Swing components.
