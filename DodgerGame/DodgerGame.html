Dodger Game (Java)

Dodger Game is a 2D arcade-style desktop game built using Java Swing. The player controls a character and must dodge falling enemies for as long as possible while earning points over time.

How the Game Works

  • The player moves left and right using keyboard controls
  • Enemies continuously fall from the top of the screen
  • The score increases the longer the player survives
  • Collision with an enemy triggers a game-over state
  • The game can be restarted without restarting the program

Tech Stack

Language: Java

Graphics: Java Swing

Concepts: Object-Oriented Programming, Game Loop, Collision Detection, Event Handling

What I Learned

  • How to build a real-time game loop in Java
  • Handling keyboard input and focus in Swing
  • Implementing collision detection using bounding boxes
  • Structuring a project using object-oriented design
  • Debugging and iterating on a graphical application
