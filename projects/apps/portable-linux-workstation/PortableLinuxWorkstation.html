Portable Linux Workstation

Created a fully functional, persistent Linux environment installed on a USB drive, allowing the operating system to boot and run independently on multiple machines.

Key Features

  • Persisted storage across reboots
  • Boots on different hardware via UEFI/BIOS
  • Custom user environment and configurations
  • Portable and isolated OS for testing and learning

What I Learned

  • Linux installation and partition
  • Bootloader configuration
  • Hardware compatibility and driver handling
  • Troubleshooting OS-level issues

Tools and Technologies

  • Ubuntu Server
  • USB Boot Tools - Ventoy
  • Filesystems
